# 格式
## godot
- 内部类名已大写开头 无下划线
- 类属性是小写开头 有下划线
- 内部方法名 小写 + 下划线 以下划线开头

## 自定义区分于godot
- 文件名 小写 + 下划线
- 类名 大写开头 无下划线 跟godot一致
- 类属性 大写开头
- 场景节点变量 以 m_ 开头 小写
- 私有方法名 小写+下划线
- 公共方法 p_ 开头 小写

# 文件结构
 - assets 资源文件夹
 - globals 全局脚本文件夹
 - objects 游戏对象文件夹
 - secenes 场景文件夹
 - models 实体对象文件夹
 - managers 游戏管理器文件夹
 - datas 游戏数据资源文件夹
 - components 组件文件夹
 - treses tres资源文件夹
